# Shosho — Appointment Management Platform

## Project Overview
Hebrew RTL SaaS for Israeli small businesses (beauticians, nail salons, etc.)

## Stack
- Frontend: React 18 + Vite + Tailwind CSS (RTL)
- Backend: Node.js + Express
- Database: Azure SQL
- Hosting: Azure Static Web Apps (frontend) + Azure App Service (backend)

## Key URLs
- Frontend: https://polite-ground-07bc5cf03.2.azurestaticapps.net
- API: https://shosho-api.azurewebsites.net

## Design Goals
- Modern, clean, professional
- Mobile-first, fully responsive
- Hebrew RTL throughout
- Pink/rose primary color (#C2185B)
- Target users: Israeli women who own small beauty businesses

## Structure
- client/ — React frontend
- server/ — Node.js backend
